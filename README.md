# 引光 H5 Demo

心象场主页的第一个本地静态 demo。

## 运行

```bash
python3 -m http.server 5173
```

打开：

```text
http://localhost:5173/
```

## 当前范围

- 使用已生成的心象场主视觉作为首页背景。
- 复刻移动端 H5 首页结构：品牌区、章节线、引光/雾墙/现实入口热点、噪流信息、底部现实取样卡片。
- 已实现 `引光` 热点聚焦态：
  - 点击引光后镜头推进。
  - 噪流、雾墙和底部卡片弱化。
  - 出现引光碎片面板。
  - 支持选择引光碎片。
  - 支持简单编辑引光名称。
  - 确认后更新底部现实回合入口。
